# Astro Swiss Assets

Assets visuels pour les emails Astro-Swiss (pictos, tarot).

## Structure

- `pictos/CARDS/` — icones avec fond carre colore (icon boxes email)
- `pictos/SOLO/` — icones seules, fond transparent
- `tarot/` — 22 arcanes majeurs (droit)
- `tarot/reversed/` — 22 arcanes majeurs (renverse)
