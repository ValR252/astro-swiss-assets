# Astro Swiss Assets

Assets visuels pour [astro-swiss](https://astro-swiss.com) — thème astral par email.

## Structure

| Dossier | Contenu | Fichiers |
|---------|---------|----------|
| `01_occidental/` | Planètes & signes (astrologie tropicale) | |
| `02_vedique/` | Grahas, rashis, nakshatras (astrologie sidérale) | |
| `03_tarot/` | 22 arcanes majeurs du Tarot de Marseille | |
| `04_numerologie/` | Chiffres 1-9 | |
| `05_pictos/` | Pictogrammes isolés (SVG + PNG) | |
| `06_elements/` | Feu, Terre, Air, Eau | |
| `07_symboles/` | Symboles sacrés (roue, lotus, om, etc.) | |

## Usage

Les images sont optimisées pour l'email (200px max, compression 50%).

URL de base : `https://raw.githubusercontent.com/ValR252/astro-swiss-assets/main/`

Exemple : `https://raw.githubusercontent.com/ValR252/astro-swiss-assets/main/07_symboles/om-vedic-nobg.png`
